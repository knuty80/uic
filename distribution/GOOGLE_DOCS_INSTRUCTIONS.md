# How to Import into Google Docs

If you are a Board Member or UIC who wants to customize this handbook without using Git or Python, follow these steps:

## Step 1: Copy the Content
1. Open the file **[GENERIC_HANDBOOK.md](GENERIC_HANDBOOK.md)** in this folder.
2. Select all the text and Copy it (Ctrl+C).

## Step 2: Paste into Google Docs
1. Create a new Google Doc.
2. Paste the text (Ctrl+V).
3. **Important:** Google Docs will automatically convert most of the formatting (Headings, Lists, Tables) from the Markdown text.

## Step 3: Global Find & Replace
Use Google Docs' **Find & Replace** (Ctrl+H) to quickly customize the document:
- Find `[League Name]` -> Replace with your actual league name.
- Find `[Abbreviation]` -> Replace with your league's short name (e.g., CMLL).
- Find `[RefTown Association ID]` -> Replace with your ID from RefTown.
- Find `$[Rate]` -> Replace with your actual pay rates.

## Step 4: Final Polishing
- **Page Breaks:** Add Page Breaks (Ctrl+Enter) before major sections (e.g., Section 3, 4, 8) to make it look professional.
- **Save as PDF:** Once you are ready, go to **File -> Download -> PDF Document**.

## Why use this method?
- **Collaboration:** You can share the Google Doc with your board for peer review.
- **No Coding:** No need to understand GitHub, Python, or Markdown syntax.
- **Mobile Access:** You can read and edit the handbook from your phone at the fields.
