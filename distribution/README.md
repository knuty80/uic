# Handbook Distribution Folder

This folder contains pre-rendered, generic versions of the UIC Handbook and Cheat Sheets. These are designed for users who do not want to use Git, Python, or the command line.

## Files included:
- `GENERIC_HANDBOOK.md`: The full handbook with empty brackets (e.g., `[League Name]`) for manual editing.
- `GOOGLE_DOCS_INSTRUCTIONS.md`: A guide on how to copy this content into a Google Doc and keep the formatting.
- `PRINTABLE_Umpire_Cheat_Sheet.html`: A generic version of the quad-fold guide.
